# AmazonConnectSDK

## Getting Started

On the root of the package (do not run in subfolders)

```
bb install
```

## Build Projects

```
bb run core build
bb run app build
bb run agent build
```

(Can also be run with npm instead of bb for a faster build)

## Publish Packages

```
bb publish -w core
bb publish -w app
bb publish -w agent
```
